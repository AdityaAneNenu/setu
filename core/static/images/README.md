# Logo Images

This directory contains the SETU logo images for dark mode and light mode themes.

## Required Files

Please place the following image files in this directory:

1. **dark-mode.png** - Logo to be displayed when the dashboard is in dark mode
2. **light-mode.png** - Logo to be displayed when the dashboard is in light mode

## Image Requirements

- Format: PNG (with transparency recommended)
- Recommended size: 40x40 pixels or larger (will be scaled to fit)
- The images will automatically switch based on the theme selection

## Setup Instructions

1. Save "dark mode.png" as "dark-mode.png" in this directory
2. Save "light mode.png" as "light-mode.png" in this directory
3. Refresh your browser to see the new logos

The logo will automatically switch between dark and light versions when users toggle the theme using the sun/moon icon in the navigation bar.
