How It Works
Post Office Route
  1. Villager writes/records complaint
  2. Postman collects and sends to PM-AJAY office
  3. Officer scans document, AI analyzes problem
  4. Worker assigned to solve issue
  5. Report sent back to villager via post office
  6. Case closed when villager confirms satisfaction
Survey Agent Support
  1. Agent visits village twice per month
  2. Records complaints for illiterate villagers
  3. Collects geotagged photos as evidence
  4. Updates on existing complaint progress
  5. Gets signatures for case closure
